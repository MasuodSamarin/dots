## Vim
```
wget -O ~/.vimrc https://raw.githubusercontent.com/txgvnn/dots/master/.vimrc
```
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

Then install plugins

```
vim +PluginInstall
```
